Task Name Task ID Parent Class Description Notes
Load Atoms loadAtoms Geometry Interface Load Atoms in-between tasks This will open a popup requesting a file name and wait for user input. Use this in combination with Save Atoms to run external commands.
Save Atoms saveAtoms Geometry Interface Save atoms in-between tasks Use this to save the current state of the Atoms in-between tasks, or in combination with Load Atoms to run external commands.
Check Geometry checkGeometry Geometry Interface Perform Atom Checks and Residue Checks on Atoms in Geometry Interface. Checks are defined in Settings/Flow.xml and can be used to provide feedback and validate geometries.
Get Specified Chain Atoms getChain Cleaner Get only the atoms in the chain specified in settings. This is essential for many external programs that can only deal with one chain at a time, otherwise chains will be dealt with individually.
Standardise Water Residues standardiseWaters Cleaner Give all water residues their standard Residue Names and PDBIDs. Standard names are defined in Settings/Atoms.xml
Strip PDB Special Characters removePDBSpecialCharacters Cleaner Remove special characters from PDB Names within residues. Also removes duplicates that might be created.
Merge Neighbouring Non-Standard Residues mergeNonStandardResidues Cleaner Merge Non-Standard Residues if they are touching. This is useful in cases where standard amino acid names are given to parts of a non-standard group e.g. 1GFL.
Calculate AMBER Types (Antechamber) calculateAMBERTypesWithAntechamber Cleaner Calculate AMBER Types using Antechamber This is currently the recommended way of computing AMBER types.
Calculate AMBER Types calculateAMBERTypes Cleaner Look up AMBER Types Experimental method that uses connectivity to determine AMBER type.
Calculate AMBER Parameters (parmchk2) calculateParameters Cleaner Use parmchk2 to calculate AMBER parameters This requires the geometry to have AMBER types already set.
Calculate Connectivity calculateConnectivity Cleaner Compute bond orders by residue Computes n^2 connectivity residue by residue, then the connections between them. This might be extremely slow if residues are huge.
Clear Connectivity clearConnectivity Cleaner Remove all bonds.
Flag Non-Standard Residues selectNonStandardResidues Cleaner Mark residues as non-standard Allows manual selection of Non-Standard Residues if they haven't been automatically detected.
Protonate Standard Residues (PDB2PQR) protonateWithPDB2PQR Protonator Protonate and assign partial charges to Standard Residues using PDB2PQR This is currently the recommended way to protonate Standard Residues, and provides control over pH.
Protonate Non-Standard Residues (Reduce) protonateWithReduce Protonator Protonate Non-Standard Residues using Reduce Experimental protonator for Non-Standard Residues. Manual protonation is recommended
Get Standard Residues getSRs Protonator Separate and cap Standard Residues. This is used to isolate Standard Residues from Atoms, allowing the use of external commands that only work on Standard Residues.
Get Non-Standard Residues getNSRs Protonator Separate and cap Non-Standard Residues. This is used to isolate Non-Standard Residues from Atoms, reducing the demand of specialist and expensive external commands.
Calculate Partial Charges (R.E.D.) calculatePartialCharges Partial Charge Calculator Compute partial charges for each residue. This requires a full optimisation of the current geometry, and takes a very long time for systems larger than 100 atoms.
Get Partial Charges from MOL2 file getPartialChargesFromMol2 Partial Charge Calculator Read partial charges from a previously completed RED calculation. This is recommended if you are using the same Non-Standard Residues from a previous calculation.
Merge Geometries mergeGeometries Geometry Merger Merge two geometries, removing capping residues. This is used to join Non-Standard and Standard Residues. Requires the geometries to be properly aligned.
Validate Layers validateLayers Gaussian Calculation Setup Check this geometry has the correct number of ONIOM Layers for a Gaussian calculation. Use this to ensure the Atoms have the correct Layer setup.
Gaussian Calculation Setup setupCalculation Gaussian Calculation Setup Get information to initialise a Gaussian calculation. Creates a popup to fill in information for a .com or .gjf file
Get Model Layer getModelLayer Gaussian Calculation Setup Isolate the Model Layer for parameterisation. This also adds Link atoms so parameters at the interface are computed.
Get Intermediate Layer getIntermediateLayer Gaussian Calculation Setup Isolate the Intermediate Layer for parameterisation. This also adds Link atoms so parameters at the interface are computed.
